# typeset-seminar-note action
## `latex-dir`
directory name where the LaTeX file to typeset exists
required: True
## `latex-filename`
name of the LaTeX file to typeset
required: True
## `jobname`
name of the PDF file
required: True
## `outdir`
directory to put the PDF file
required: True
## `max-iterations`
number of max iterations to run latex
required: False
default: 3
